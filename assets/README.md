# assets

Placeholder brand assets — **replace these** with your own (see
[../docs/BRANDING.md](../docs/BRANDING.md)):

- `icon.png` — 1024×1024 app icon
- `splash.png` — splash image, centered on your `brand.json` `colors.bg`

The shipped placeholders are solid `#0b1020` fills so the project builds out of
the box.
